# @mjolnir/tsconfig
Shared TypeScript config for @mjolnir.

## install
`yarn add @mjolnir/tsconfig -D`

or

`npm i @mjolnir/tsconfig -D`

## useage
add `extends` in your tsconfig.json:

```json
{
  "extends": "@mjolnir/tsconfig",
  ...
}
```


### react env

```json
{
  "extends": "@mjolnir/tsconfig/react.json"
}
```
